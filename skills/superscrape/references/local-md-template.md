# Superscraper Local Configuration

This file stores per-machine settings for the Superscraper plugin.
It is created automatically during first deploy and should not be committed to git.

## Fields

- **vps_host** — IP address of your VPS for Streamlit deploy
- **vps_user** — SSH username (usually root)
- **ssh_key_configured** — true after SSH key is set up
- **github_user** — GitHub username for Pages deploy
- **firecrawl_configured** — true after Firecrawl CLI is authenticated
